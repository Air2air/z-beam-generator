# Visual Appearance Research Prompt

## Context
Generate detailed visual appearance characteristics for {{contaminant_name}} contamination.

WORD LENGTH: 100-300 words

## Task
Research the visual appearance of {{contaminant_name}} on surfaces. Provide specific details about:

1. Color (specific shades, variations, aging changes)
2. Texture (smooth, rough, flaky, powdery, crystalline, etc.)
3. Thickness (typical range in micrometers)
4. Pattern (uniform, patchy, concentrated, streaking)
5. Surface effects (glossy, matte, iridescent, dull)

## Output Format
Structured description with specific technical details:
```
Color: [specific color description]
Texture: [specific texture description]
Thickness: [range] micrometers
Pattern: [distribution pattern description]
Surface: [surface effect description]
```

## Requirements
- Be specific and technical (not generic)
- Include measurable characteristics
- Describe how appearance varies with age/conditions
- NO FALLBACKS - Research specific to this contamination
